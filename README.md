# wash-module-dynamic-pricing

Модуль WASH PRO CRM: **Динамические цены**.

Повышает цены постов при высокой занятости выбранной автомойки и возвращает исходные цены при снижении нагрузки.

## Настройки

| Ключ | Описание |
|------|----------|
| `wash_id` | ID автомойки |
| `busy_threshold` | Минимум занятых постов для повышения (например, 9) |
| `price_increase_percent` | Процент повышения для режимов с ценой > 0 |
| `poll_interval` | Интервал опроса (сек) |
| `api_login` / `api_password` | Учётные данные CRM для записи цен |

## Запуск

1. Установите модуль из каталога CRM.
2. Откройте настройки, выберите автомойку и порог занятости.
3. Запустите модуль (PyOrchestrator).
